# API Security Assignment

This project demonstrates secure API usage using environment variables, proper error handling, and privacy protection.

## Security & Privacy Answers

Exposing an API key on GitHub can lead to unauthorized usage, financial loss, and service abuse. Attackers can consume API limits or perform malicious actions using the key.

Logging city names is restricted because location data is considered sensitive personal data under privacy laws like GDPR. Storing or logging such data without consent violates data minimization and user privacy principles.
